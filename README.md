# Git Tools

Características

- ✅ Instalación con un solo comando
- ✅ Descarga automática de TODOS los scripts disponibles
- ✅ Fácil de mantener y actualizar
- ✅ Compatible con PHPStorm y otras IDEs


**Instalación Rápida**

» En la raíz de tu proyecto:
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/miguelcastanedaV/git_tools/main/install.sh)
```

**Desinstalación Rápida**

» En la raíz de tu proyecto:
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/miguelcastanedaV/git_tools/main/uninstall.sh)
```
